# Cloudflare Pages Configuration

Clef uses two Cloudflare Pages projects. The JSON files
in this directory are the canonical record of project
settings. Apply them manually via the Cloudflare dashboard
— they cannot be applied automatically.

| Project   | Domain       | Source |
| --------- | ------------ | ------ |
| clef-site | clef.sh      | www/   |
| clef-docs | docs.clef.sh | docs/  |

For setup instructions see:
docs/contributing/deployment.md
